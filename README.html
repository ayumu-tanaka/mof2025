講義概要・講義目的

講義概要

現代のグローバル経済について経済学の視点から講義し、国際貿易論の考え方を解説します。伝統的貿易理論（リカード・モデル、ヘクシャー＝オリーン・モデル）、新々貿易理論、貿易政策論といった標準的な内容をカバーします。

講義目的

経済学の視点から現代のグローバル化を理解できるようになることを目標とします。
